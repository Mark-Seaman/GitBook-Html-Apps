### Create simple page

Add this content to index.html

```html
    <!doctype html>
    <html>
        <head>
            <title>Learning HTML</title>
        </head>
        <body>
            <h1>Core HTML elements</h1>
            <p>Paragraph - p</p>
            <p>Title - title</p>
            <p>Top Heading - h1</p>
            <p>Subheading - h2</p>
        </body>
    </html>
```
