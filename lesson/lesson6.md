## Tables in HTML

### Table Tag
* Define a table
```html
<table>
</table>
```

### Table Row
* Define a series of rows in the table

```html
<table>

    <tr></tr>
    <tr></tr>
    <tr></tr>

</table>
```


### Table Columns
* Define a series of rows in the table

```html
<table>

    <tr>
        <td>Column #1</td>
        <td>Column #2</td>
    </tr>

</table>
```

### Table Headers
* Define column headers in the table

```html
<table>
    <tr>
        <th>Name</th>
        <th>Address</th>
    </tr>
    <tr>
        <td>Mark Seaman</td>
        <td>PO Box 14</td>
    </tr>
    <tr>
        <td>Bill Gates</td>
        <td>Redmond, WA</td>
    </tr>
</table>
```
