Auto Page Redirect

This page demonstrates how to make a page that jumps automatically to a different page when loaded.

Add the following text to the end of the body in the page.

    <script>
        window.location.href = "http://bacs200.unco.edu/BACS_200/index.html";
    </script>

Full Page HTML

Load a different page

<!doctype html>
<html lang="en">

    <head>
        <meta charset="utf-8">
    </head>

    <body>
        <h1>BACS 200 Redirect</h1>
        <p>
            If you are not redirected within 3 seconds then go to
            <a href="http://bacs200.unco.edu/BACS_200/index.html">http://bacs200.unco.edu/BACS_200/index.html</a>
        </p>

        <script>
            window.location.href = "http://bacs200.unco.edu/BACS_200/index.html";
        </script>
    </body>

</html>
