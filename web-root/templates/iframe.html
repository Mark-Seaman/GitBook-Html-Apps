Problem - If you place HTML in your file then the browser will render it as HTML. What if you want to display the HTML source rather than the formatted output?

Solution 1 - The proper solution is to replace the characters in HTML for display.

Replace the & character with &
Replace the < character with <
Replace the > character with >

Solution 2 - A better solution is to use an iframe for display of HTML. Load the HTML from a text file.
