CSS Selector Workshop

Uniform selectors

Reset the default styles used by the browswer 
                * { margin: 10px; }

                main { padding: 1em; }

                h1 { color:blue; }

Element Selectors

Select elements by element type 
                < h1 class="idea" >Headline< /h1 >
            
                idea { color: green; }

Class Selectors

Apply styles to different types of elements 
                < h1 class="button" >Headline< /h1 >
                < p class="button" >Paragraph< /p >
            
                .button {
                    background-color: green;
                    color: white
                }

ID Selectors

Select specific elements 
                < h1 id="headline" >
                    Headline
                < /h1 >
                < nav id="main-menu" > < /nav >
            
                #headline {
                    background-color: green;
                    color: white
                }

                #main-menu {
                    display: inline;
                }

Element and Class Selector

Select specific elements with certain classes 
                < header >
                    < h1 id="headline" >
                        Headline
                    < /h1 >
                < /header >
            
                header.h1 {
                    background-color: green; color: white
                }

Multiple Selectors

Select multiple element types 
                < header >
                    < h1 id="headline" >
                        Headline
                    < /h1 >
                < /header >
            
                h1,h2,h3 {
                    background-color: green;
                    color: white
                    font-family: sans-serif;
                }

Psuedo-class CSS Selectors

Control styles on hyperlinks 
                < a href="http://google.com">
                    Google
                < /a >
            
                a:link {
                    color: green;
                }

                a:visited {
                    color: blue;
                }

                a:hover {
                    color: red;
                }
